# 🍎 Apple Logo Keychain

A 3D Apple logo keychain designed using Onshape as part of a Computer-Aided Design (CAD) assignment.

---

## 📖 Project Overview

This project recreates the Apple logo as a 3D printable keychain. The model was created from a sketch in Onshape, converted into a solid body, and prepared for 3D printing.

---

## ✨ Features

- 🍎 Apple logo design
- 📏 2 mm extrusion thickness
- 🔵 4 mm keychain hole
- 🖨️ Exported as an STL file
- 💻 Designed entirely in Onshape
